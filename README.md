A simple tool to calibrate an image plane for calibrated sketch building when reverse engineering parts.

## Features
- GUI dialog
- Saves calibration data for later reference/undo.

## Installation
1. Open FreeCAD
2. Go to **Tools → Addon Manager**
3. Search for "CalibrateImage"
4. Click **Install**

Or install manually by copying the folder to your `Mod` directory.

## Requirements
- FreeCAD 1.0 or newer
